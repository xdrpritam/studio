# **App Name**: UnMac

## Core Features:

- MAC Address Unblock Request Form: Users can input their MAC Address, Device Name, select WiFi Provider (Jio, Airtel, BSNL), enter WiFi Name (SSID), and choose a plan (Free Trial or Paid).
- Trial Period Activation: Allows users to activate a 15-minute free trial for unblocking. Displays a countdown timer upon activation.
- Secure Payment System: Integrate a payment gateway for the ₹100/month paid plan. Displays a 'Upgrade to Premium' button and shows a payment success confirmation page.
- User Dashboard: After submission, displays submitted details, unblock status ('Processing' / 'Unblocked'), a countdown timer for trial users, and plan validity for paid users.
- Basic Backend Processing & Storage: Stores user input in a database, validates MAC address format, and handles the logic for trial vs. paid plans. Designed for future API integration.
- Smart Troubleshooting Assistant: An AI-powered tool that analyzes submitted details like device name and WiFi provider to suggest context-aware troubleshooting steps or reasons for network blocking.
- Informational Content Pages: Static 'About Us', 'FAQ', and a 'Contact Form' for user inquiries.

## Style Guidelines:

- Primary interactive color: Vivid purple-blue (#6155FF) for active elements and highlights, embodying a futuristic tech vibe.
- Background color: A very dark, subtle purple-black (#17151A) to provide a deep, immersive canvas for the neon accents.
- Accent color: A vibrant sky blue (#26A9E3) used for secondary interactive elements, providing dynamic contrast and reinforcing the 'neon' aesthetic.
- Headline font: 'Poppins' (sans-serif) for its modern, geometric appearance, ideal for strong, impactful headings. Note: currently only Google Fonts are supported.
- Body font: 'Inter' (sans-serif) for its clean, neutral readability, ensuring comfortable reading of longer text blocks like FAQs. Note: currently only Google Fonts are supported.
- Use clean, minimalistic line icons that can easily integrate glow effects to complement the neon tech aesthetic.
- Responsive layout optimized for various screen sizes, featuring a prominent hero section with main heading 'UnBlock Your Mac Addresses With UnMac' and a clear call-to-action.
- Smooth, subtle animations on page transitions, element interactions, and hover effects on rounded buttons to enhance user engagement and provide a polished, dynamic feel.
